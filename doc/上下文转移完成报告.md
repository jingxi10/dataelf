## 📝 关键信息总结

### 管理后台访问地址

#### 开发环境
```
http://localhost:3000/admin
```

#### 生产环境
```
https://your-domain.com/admin
```

### 访问要求
1. **角色**：必须是管理员（ADMIN）
2. **状态**：账号必须是正常状态（ACTIVE）
3. **认证**：需要有效的登录Token

### 三种访问方式
1. **直接访问**：在浏览器输入完整URL
2. **导航菜单**：登录后点击头像 → 选择"管理后台"
3. **相对路径**：直接访问 `/admin` 路径

### 首次使用步骤
1. 创建管理员账号（数据库设置）
   ```sql
   UPDATE users SET role = 'ADMIN' WHERE email = 'admin@example.com';
   ```
2. 登录系统
3. 访问管理后台
