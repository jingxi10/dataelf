# 第一部分：快速开始

## 1.1 系统访问

### 访问地址
- **前端地址**: http://localhost:3000
- **后端API**: http://localhost:8080
- **API文档**: http://localhost:8080/swagger-ui.html

### 系统要求
- **浏览器**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **屏幕分辨率**: 最低 1280x720
- **网络**: 稳定的互联网连接

## 1.2 账号注册

### 注册步骤
1. 访问首页，点击右上角"注册"按钮
2. 填写注册信息：
   - **邮箱**: 用于接收通知和找回密码
   - **手机号**: 用于账号验证
   - **密码**: 至少8位，包含字母和数字
3. 点击"注册"提交
4. 等待管理员审核

### 注册后
- 系统提示：**"注册成功，请等待审核"**
- 邮箱会收到注册确认邮件
- 审核通过后会收到通知邮件
- 审核时间：通常1-2个工作日

## 1.3 账号登录

### 登录步骤
1. 点击"登录"按钮
2. 输入邮箱和密码
3. 点击"登录"

### 登录状态
- **审核中**: 提示"账号正在审核中"
- **已过期**: 提示"账号已过期，请联系管理员"
- **正常**: 成功登录，跳转到首页

# 第二部分：用户指南

## 2.1 浏览内容

### 首页浏览
- **最新内容**: 显示最新发布的内容
- **热门内容**: 按浏览量排序
- **分类浏览**: 通过分类筛选内容
- **标签云**: 点击标签查看相关内容

### 内容详情
1. 点击内容标题进入详情页
2. 查看完整内容
3. 查看作者信息
4. 查看发布时间和浏览量

### 搜索功能
1. 在搜索框输入关键词
2. 按回车或点击搜索按钮
3. 查看搜索结果
4. 可按分类、标签筛选

## 2.2 内容互动

### 点赞
1. 在内容详情页点击"点赞"按钮
2. 再次点击取消点赞
3. 需要登录才能点赞

### 收藏
1. 点击"收藏"按钮
2. 内容保存到"我的收藏"
3. 在个人中心查看收藏列表
4. 需要登录才能收藏

### 评论
1. 在内容详情页底部找到评论区
2. 输入评论内容
3. 点击"发表评论"
4. 需要登录才能评论

### 分享
1. 点击"分享"按钮
2. 选择分享平台：
   - **微信**: 扫描二维码分享
   - **微博**: 跳转到微博分享页
   - **QQ空间**: 跳转到QQ空间
   - **Twitter**: 跳转到Twitter
   - **Facebook**: 跳转到Facebook
   - **LinkedIn**: 跳转到LinkedIn
   - **复制链接**: 一键复制分享链接

## 2.3 个人中心

### 我的收藏
1. 点击顶部导航"我的收藏"
2. 查看所有收藏的内容
3. 点击内容查看详情
4. 点击"取消收藏"移除

### 我的内容
1. 点击顶部导航"我的内容"
2. 查看自己创建的所有内容
3. 按状态筛选：
   - **全部**: 所有内容
   - **草稿**: 未提交的内容
   - **待审核**: 已提交等待审核
   - **已发布**: 审核通过已发布

### 内容操作
- **编辑**: 修改内容（仅草稿和被退回的内容）
- **提交审核**: 提交草稿到审核队列
- **发布**: 审核通过后发布（管理员操作）
- **删除**: 删除草稿或被退回的内容

### 通知中心
1. 点击顶部通知图标
2. 查看系统通知：
   - 审核通知
   - 评论通知
   - 点赞通知
   - 系统公告
3. 点击通知查看详情
4. 标记为已读

## 2.4 创建内容

### 进入编辑器
1. 登录后点击"创建内容"
2. 选择内容模板：
   - 技术文章模板
   - 案例研究模板
   - 数据报告模板
   - 产品评测模板
   - 调查报告模板
   - 行业分析模板

### 填写内容

#### 基本信息
- **标题**: 简洁明了，50字以内
- **摘要**: 内容概述，200字以内
- **封面图**: 上传封面图片（可选）

#### 内容编辑
- **富文本编辑器**: 支持格式化文本
- **Markdown**: 支持Markdown语法
- **图片上传**: 插入图片
- **代码块**: 插入代码

#### 分类和标签
- **选择分类**: 从分类树中选择（最多3级）
- **添加标签**: 输入标签，回车添加（最多10个）
- **标签建议**: 系统自动推荐相关标签

#### 结构化字段
根据选择的模板，填写对应的结构化字段：
- 技术文章：技术栈、难度等级、代码示例
- 产品评测：产品信息、评分、优缺点
- 数据报告：数据来源、统计方法、图表

### 保存和提交
- **保存草稿**: 保存但不提交审核
- **提交审核**: 提交到审核队列
- **预览**: 预览发布效果

### 审核流程
1. **提交审核**: 内容进入审核队列
2. **管理员审核**: 
   - 通过：内容发布
   - 退回：返回草稿，附带修改意见
3. **收到通知**: 邮件和站内信通知审核结果

# 第三部分：管理员指南

## 3.1 管理后台访问

### 登录管理后台
1. 使用管理员账号登录
2. 点击右上角头像
3. 选择"管理后台"
4. 或直接访问 `/admin`

### 管理后台布局
- **左侧菜单**: 功能导航
- **顶部栏**: 页面标题、通知、用户信息
- **主内容区**: 管理功能界面

## 3.2 用户管理

### 访问用户管理
1. 点击左侧菜单"用户管理"
2. 查看所有用户列表

### 用户列表
显示信息：
- ID、邮箱、手机号
- 角色（用户/管理员）
- 状态（待审核/正常/已过期/已禁用）
- 注册时间、过期时间

### 用户操作

#### 审核用户
1. 找到状态为"待审核"的用户
2. 点击"审核"按钮
3. 选择操作：
   - **通过**: 设置账号有效期，用户可正常使用
   - **拒绝**: 拒绝注册申请

#### 设置有效期
1. 点击"设置有效期"
2. 选择有效期：
   - 1个月
   - 3个月
   - 6个月
   - 1年
   - 永久
3. 确认设置

#### 禁用/启用用户
1. 点击"禁用"按钮
2. 确认禁用
3. 用户无法登录
4. 点击"启用"恢复

#### 删除用户
1. 点击"删除"按钮
2. 确认删除
3. 用户及其数据被永久删除

### 批量操作
1. 勾选多个用户
2. 选择批量操作：
   - 批量审核
   - 批量禁用
   - 批量删除

## 3.3 模板管理

### 访问模板管理
1. 点击左侧菜单"模板管理"
2. 查看所有内容模板

### 模板列表
显示信息：
- 模板名称、类型
- 字段配置
- 使用次数
- 创建时间

### 模板操作

#### 创建模板
1. 点击"创建模板"
2. 填写模板信息：
   - **名称**: 模板名称
   - **类型**: 模板类型
   - **描述**: 模板说明
3. 配置字段：
   - 添加字段
   - 设置字段类型
   - 设置必填/可选
4. 保存模板

#### 编辑模板
1. 点击"编辑"按钮
2. 修改模板配置
3. 保存更改

#### 删除模板
1. 点击"删除"按钮
2. 确认删除
3. 注意：使用中的模板无法删除

## 3.4 内容审核

### 访问内容审核
1. 点击左侧菜单"内容审核"
2. 查看待审核内容列表

### 审核列表
显示信息：
- 内容标题、作者
- 模板类型
- 提交时间
- 状态

### 审核操作

#### 查看内容
1. 点击内容标题
2. 查看完整内容
3. 检查内容质量：
   - 内容完整性
   - 格式规范性
   - 结构化数据
   - 是否违规

#### 通过审核
1. 点击"通过"按钮
2. 内容自动发布
3. 作者收到通过通知

#### 退回修改
1. 点击"退回"按钮
2. 填写退回原因
3. 内容返回草稿状态
4. 作者收到退回通知

#### 批量审核
1. 勾选多个内容
2. 点击"批量通过"或"批量退回"
3. 确认操作

### 审核建议
- 检查内容是否符合平台规范
- 验证结构化数据完整性
- 确保内容质量
- 及时处理审核队列

## 3.5 分类管理

### 访问分类管理
1. 点击左侧菜单"分类管理"
2. 查看分类树

### 分类树
- **树形展示**: 显示3级分类结构
- **展开/收起**: 控制树形展开状态
- **图标区分**: 不同层级使用不同图标

### 分类操作

#### 创建根分类
1. 点击"创建根分类"
2. 填写信息：
   - **名称**: 分类名称（2-50字符）
   - **描述**: 分类说明（可选）
   - **排序**: 数字越小越靠前
3. 点击"创建"

#### 创建子分类
1. 悬停在父分类上
2. 点击"+"按钮
3. 填写子分类信息
4. 点击"创建"

**注意**: 最多支持3级分类

#### 编辑分类
1. 点击分类节点
2. 在详情抽屉中点击"编辑"
3. 修改分类信息
4. 点击"更新"

**注意**: 层级不能修改

#### 删除分类
1. 点击分类节点
2. 点击"删除"按钮
3. 确认删除

**注意**: 删除父分类会同时删除所有子分类

### 分类详情
查看信息：
- 分类名称、层级
- 排序、内容数量
- 子分类数量
- 创建时间

## 3.6 标签管理

### 访问标签管理
1. 点击左侧菜单"标签管理"
2. 查看标签列表和标签云

### 标签云预览
- **可视化展示**: 标签大小和颜色根据使用量变化
- **点击交互**: 点击标签查看相关内容
- **使用次数**: 显示每个标签的使用次数

### 标签列表
显示信息：
- ID、标签名称
- 使用次数
- 创建时间

### 标签操作

#### 创建标签
1. 点击"创建标签"
2. 输入标签名称（2-50字符）
3. 点击"创建"

**注意**: 标签名称必须唯一

#### 搜索标签
1. 在搜索框输入关键词
2. 实时显示搜索结果
3. 清空搜索框恢复全部

#### 排序标签
切换排序方式：
- **按使用量**: 最热门的标签在前
- **按名称**: 字母顺序
- **按时间**: 最新创建的在前

#### 查看标签内容
1. 点击标签云中的标签，或
2. 点击列表中的"查看内容"
3. 在抽屉中查看相关内容列表
4. 点击内容跳转到详情页

#### 删除标签
1. 找到使用次数为0的标签
2. 点击"删除"按钮
3. 确认删除

**注意**: 只能删除未被使用的标签

## 3.7 数据源管理

### 访问数据源管理
1. 点击左侧菜单"数据源管理"
2. 查看数据源列表

### 数据源列表
显示信息：
- ID、名称、URL
- 类型、状态
- 抓取间隔
- 抓取统计（总次数/成功/失败）
- 最后抓取时间

### 数据源操作

#### 添加数据源
1. 点击"添加数据源"
2. 填写基本信息：
   - **名称**: 数据源名称
   - **URL**: 数据源地址
   - **类型**: 选择类型（RSS/HTML/API/JSON/XML）
   - **抓取间隔**: 设置间隔（1-168小时）
   - **描述**: 数据源说明（可选）

3. 配置高级选项（可选）：
   - **选择器配置**: JSON格式，配置数据提取规则
   - **清洗规则**: JSON格式，配置数据清洗规则
   - **模板映射**: JSON格式，配置数据到模板的映射

4. 点击"创建"

#### 编辑数据源
1. 点击"编辑"按钮
2. 修改配置信息
3. 点击"更新"

#### 查看详情
1. 点击"查看"按钮
2. 在抽屉中查看完整信息：
   - 基本信息
   - 抓取统计
   - 时间信息
   - 错误信息（如有）

#### 手动抓取
1. 确保数据源已启用
2. 点击"抓取"按钮
3. 系统立即触发抓取任务

#### 启用/禁用
1. 点击"启用"或"禁用"按钮
2. 禁用后停止自动抓取
3. 启用后恢复自动抓取

#### 删除数据源
1. 点击"删除"按钮
2. 确认删除
3. 数据源及配置被永久删除

### 数据源类型说明

#### RSS订阅
- **适用**: 博客、新闻网站的RSS feed
- **示例**: `https://example.com/feed.xml`

#### HTML网页
- **适用**: 普通网页内容抓取
- **示例**: `https://example.com/articles`
- **需要**: 配置选择器

#### API接口
- **适用**: RESTful API数据源
- **示例**: `https://api.example.com/v1/articles`

#### JSON数据
- **适用**: JSON格式的数据文件
- **示例**: `https://example.com/data.json`

#### XML数据
- **适用**: XML格式的数据文件
- **示例**: `https://example.com/data.xml`

### 抓取间隔建议
- **新闻网站**: 1-6小时
- **博客**: 12-24小时
- **学术期刊**: 24-168小时
- **API接口**: 根据限制设置

### 访问系统设置

# 第四部分：常见问题

## 4.1 账号相关

### Q1: 注册后多久能使用？
**A**: 注册后需要等待管理员审核，通常1-2个工作日。审核通过后会收到邮件通知。

### Q2: 忘记密码怎么办？
**A**: 
1. 点击登录页面的"忘记密码"
2. 输入注册邮箱
3. 查收重置密码邮件
4. 点击邮件中的链接重置密码

### Q3: 账号过期了怎么办？
**A**: 联系管理员延长账号有效期。

### Q4: 可以修改邮箱吗？
**A**: 目前不支持自行修改邮箱，请联系管理员。

## 4.2 内容相关

### Q5: 内容提交后多久能发布？
**A**: 提交后进入审核队列，管理员会在1-3个工作日内审核。审核通过后自动发布。

### Q6: 内容被退回怎么办？
**A**: 
1. 查看退回原因
2. 在"我的内容"中找到被退回的内容
3. 点击"编辑"修改
4. 重新提交审核

### Q7: 可以删除已发布的内容吗？
**A**: 已发布的内容不能自行删除，请联系管理员。

### Q8: 支持哪些内容格式？
**A**: 支持富文本、Markdown、图片、代码块等多种格式。

### Q9: 一篇内容可以添加多少个标签？
**A**: 最多可以添加10个标签。

### Q10: 如何选择合适的模板？
**A**: 根据内容类型选择：
- 技术教程 → 技术文章模板
- 实际案例 → 案例研究模板
- 数据分析 → 数据报告模板
- 产品介绍 → 产品评测模板

## 4.3 功能相关

### Q11: 如何分享内容到微信？
**A**: 
1. 点击"分享"按钮
2. 选择"微信"
3. 使用微信扫描二维码
4. 分享到微信好友或朋友圈

### Q12: 收藏的内容在哪里查看？
**A**: 点击顶部导航"我的收藏"查看所有收藏的内容。

### Q13: 如何关注其他用户？
**A**: 在用户主页点击"关注"按钮。

### Q14: 通知在哪里查看？
**A**: 点击顶部通知图标（铃铛）查看所有通知。

## 4.4 管理员相关

### Q15: 如何批量审核用户？
**A**: 
1. 在用户管理页面勾选多个用户
2. 点击"批量审核"
3. 选择操作并确认

### Q16: 数据源抓取失败怎么办？
**A**: 
1. 查看错误信息
2. 检查URL是否正确
3. 检查网络连接
4. 检查选择器配置
5. 联系技术支持

### Q17: 如何备份数据？
