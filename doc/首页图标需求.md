## 图标设计规范

### 颜色
- 主色：#1A73E8（蓝色）
- 选中状态：白色图标 + 蓝色背景
- 未选中状态：#666666（灰色）
- 悬停状态：#1A73E8（蓝色）

### 风格
- 线性图标（Line icons）
- 圆角：2px
- 线条粗细：1.5px
- 风格统一，简洁现代

### 文件命名
```
icon-category-all.svg      # 全部
icon-category-tech.svg     # 科技
icon-category-data.svg     # 数据科学
icon-category-blockchain.svg # 区块链
icon-category-arch.svg     # 数据架构
icon-category-web.svg      # Web开发
icon-category-ai.svg       # 人工智能
icon-category-cloud.svg    # 云计算
icon-like.svg              # 点赞
icon-favorite.svg          # 收藏
icon-comment.svg           # 评论
icon-search.svg            # 搜索
icon-twitter.svg           # Twitter
icon-weibo.svg             # 微博
icon-github.svg            # GitHub
```

### 存放位置
将图标文件放在 `frontend/src/assets/icons/` 目录下

---

## 备选方案

如果不想自己设计图标，可以使用以下图标库：

1. **Element Plus 内置图标** - 项目已集成，可直接使用
2. **Iconify** - 开源图标库，支持多种风格
3. **阿里巴巴 Iconfont** - 国内常用图标库
